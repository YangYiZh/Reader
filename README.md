# Reader 应用

一个基于 Flutter 开发的阅读应用，提供流畅的阅读体验和丰富的功能。

---

## 功能特性

- **多格式支持**：支持多种文件格式（如 PDF、EPUB 等）。
- **书签功能**：随时保存阅读进度。
- **夜间模式**：保护眼睛，适合夜间阅读。
- **自定义设置**：字体大小、行间距、主题颜色等可自由调整。

---

## 安装与运行

### 1. 环境要求

- **Flutter SDK**：请确保已安装 Flutter 开发环境。
- **依赖工具**：
  - Android Studio（用于 Android 构建）
  - Xcode（用于 iOS 构建）
